O plugin twiitada serve para exibir seus posts do twitter no seu blog.

== Description ==

O twittada é um plugin que serve para exibir, no seu blog, um widget que mostra os últimos posts que você fez no twitter. O Twittada possui um painel de administração onde você pode modificar o tamanho, as cores do texto e linhas, pode também definir a quantas mensagens deve aparecer de forma a adaptar o widget ao design do seu blog.

A equipe de desenvolvimento do plugin Twittada não se responsabiliza por nenhum problema ocasionado pela instalação do mesmo e, apesar de ajudar a resolver problemas com instalação, não tem obrigação de prestar suporte.

== INSTALLATION ==

Para instalar o plugin Twittada você deve seguir os passos básicos de intalação, ativação, de plugins WordPress que, basicamente, consiste no upload do plugin para a pasta wp-content/plugins do seu blog e na ativação do mesmo através do menu Plugins do painel administrativo do seu blog.

Abaixo um passo a passo de instalação e configuração do plugin Twittada:

1 - Acesse o FTP do seu blog e faça o upload da pasta twittada para o diretorio wp-content/plugins do seu blog;

	1.1 - Dentro da pasta twittada devem existir 5 arquivos: a) twittada.php; b) logo_twitter.png c) readme.txt d) leiame-txt e) sreenshot.jpg;

2 - Acesse o painel administrativo do seu blog e vá até o menu Plugins;

3 - Procure o plugin Twittada na lista e ative o plugin;

Após a instalação você deve configurar o plugin para ele começar a exibir corretamente no seu  blog.

A configuração do plugin consiste na escolha do local onde as mensagens deverão aparecer no seu blog e em informar ao sistema o nome do usuário Twitter cujas mensagens você deseja exibir no blog e modificar a largura, cor da fonte e da borda para melhor adequação ao design do seu blog.

Existem duas formas para instalar e configurar o plugin, incluindo o código diretamente no template do seu blog ou através do menu widgets do WordPress.

1 - Após ativação do plugin acesse o menu Configurações >> Twittada;

2 - Preencha o formulário com os dados solicitados e salve as opções;

3 - Para incluir no tema você pode proceder de duas maneiras diferentes:
	
	3.1 - MENU WIDGETS
		
		A - Acesse o menu Design >> Widigets;
		B - Encontre o widget Twittada e adicione na sidebar que desejar;
		C - Salve as alterações.

	3.2 - DIRETO NO TEMPLATE
	
		A - Inclua o código <?php twittada(); ?> no local onde as mensagens devem aparecer;

== FREQUENTLY ASKED QUESTIONS ==

= Instalei o twittada mas as mensagens que estão aparecendo no blog não são do meu twitter? =

Você precisa acessar o Menu Configurações >> Twittada para definir o usuário.
